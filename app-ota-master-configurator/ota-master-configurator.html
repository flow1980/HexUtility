OTA Master Configurator

Generates the configuration files (.lua files) for the OTA master (CarMediaLab's Connectivity Control Units (CCU) Flea).

An OTA master configuration consists of at least 2 files: One Update manifest file and at least one ECU update script file.
The Vehicle Pre Post Actions file is optional.
You don't have to fill out all 3 tabs. You just have to fill out the tabs for the files you want to generate!
If you want to generate multiple ECU update script files, you have to do it one by one in the 'ECU update script' tab!
