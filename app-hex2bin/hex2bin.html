Hex2Bin Converter

Converts a HEX file [Intel HEX (.hex) or Raw HEX (.hex)] to a raw binary file.

Input file (.hex file):
[OPTIONAL] Address Range(s) of the .hex file to be converted:

Syntax:
StartAddress1:EndAddress1,StartAddress2:EndAddress2,...
Addresses in hex format without prefix '0x' or postfix 'H', not longer than 32 bit and case insensitive.
Address ranges separated by comma (','), no spaces allowed at all.
As many .bin files as the number 'n' of specified address ranges are created.
The .bin files are named with a suffix from 0 to n-1.
Example:
0:3FFF,4000:1FFFF
